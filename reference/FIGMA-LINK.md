## How to Inspect (FREE — no paid account needed)

1. Open the link above in Chrome
2. You may need to sign in with a free Figma account
3. Once open, you will see the design on the canvas
4. Look for the section called **"Assessment test"** — this is what you need to build
5. You will see 3 frames: Desktop (1440px), Tablet (768px), Mobile (320px)
6. There is also a **"Global Fonts & Colors"** section — use this as your style reference
7. There is also a **"Buttons & Hover States"** section — use this for button hover styles
8. There is a **"Hover"** section that shows where hover interactions should be applied (shown with mouse cursor icons)

### To inspect any element:
- Click on the element in the design
- Look at the right panel → you will see CSS properties
- Font family, font size, font weight, line height, letter spacing
- Colors (click to copy hex code)
- Width, height, padding, margin, border-radius
- You can also see spacing between elements by selecting one element and hovering over another

### Fonts used in this design:
- **Quicksand** (Google Fonts — free): Used for headings, buttons, pre-headers
- **Lato** (Google Fonts — free): Used for body text, menu items, captions

### Colors:
Inspect the "Color Palette" component in the "Global Fonts & Colors" section for exact hex values.
Common colors you will see: the brand primary color, white, dark text, and accent colors.
Use the Figma inspect panel to get the exact hex codes from the design elements.
